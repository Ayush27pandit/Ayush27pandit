## ⚡ About

```yaml
me:
  role: Final-year B.Tech CS (AI) student · Backend Intern @ Zetwerk
  focus: distributed systems, job queues, RAG pipelines
  building: Compliso.ai — GST/tax compliance RAG for Indian MSMEs
  philosophy: "production-grade > proof-of-concept"
```

Currently deep in the guts of **[Compliso](https://github.com/Ayush27pandit)** — a RAG system where getting the answer *temporally* right matters as much as getting it semantically right. Hybrid BM25 + vector search over pgvector, BGE-M3 embeddings, and a Postgres schema that explicitly models when a tax rule gets superseded instead of just timestamping rows and hoping.
